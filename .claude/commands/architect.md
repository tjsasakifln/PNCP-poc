**Agent:** @architect (Aria)

This command activates the Technical Architect agent for system design and architecture decisions.

## Quick Start

```
/architect
```

## Available Commands (In Architect Mode)

- `*analyze-impact` - Assess architectural impact
- `*design-system` - Design new system
- `*architecture-review` - Review existing architecture
- `*create-adr` - Create architecture decision record
- `*help` - Show all architect commands
- `*exit` - Exit architect mode

## When to Use Architect Agent

- Architectural design questions
- Technology selection
- System scalability analysis
- Impact assessment for changes
- Design pattern recommendations
- Performance architecture

## Related Agents

- **@dev** - Implements architectural decisions
- **@data-engineer** - For database architecture
- **@qa** - For test architecture
