```

```

## Available Commands (In PO Mode)

- `*manage-backlog` - Manage product backlog
- `*refine-story` - Refine story acceptance criteria
- `*prioritize-backlog` - Prioritize backlog items
- `*create-prd` - Create product requirements document
- `*help` - Show all PO commands
- `*exit` - Exit PO mode

## When to Use PO Agent

- Managing product backlog
- Story refinement and acceptance criteria
- Backlog prioritization
- PRD creation
- Feature planning
- User story management
