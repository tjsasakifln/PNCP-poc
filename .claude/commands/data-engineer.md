This command activates the Data Engineer agent for database design and operations.

## Quick Start

```
/data-engineer
```

## Available Commands (In Data Engineer Mode)

- `*db-domain-modeling` - Design database schema
- `*db-apply-migration` - Execute database migration
- `*db-schema-audit` - Audit database schema
- `*db-rls-audit` - Review row-level security
- `*db-analyze-hotpaths` - Analyze query performance
- `*db-rollback` - Rollback migration
- `*help` - Show all data-engineer commands
- `*exit` - Exit data-engineer mode

## When to Use Data Engineer Agent

- Database schema design
- Migration planning and execution
- Query optimization
- RLS policy implementation
- Performance tuning
- Data architecture decisions
- Supabase configuration
