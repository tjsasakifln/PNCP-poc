**Agent:** @sm (River)

**File:** `.claude/commands/AIOS/agents/sm.md`

This command activates the Scrum Master agent for agile ceremonies and team facilitation.

## Quick Start

```
/sm
```

## Available Commands (In SM Mode)

- `*create-next-story` - Create next development story
- `*facilitate-ceremony` - Facilitate agile ceremony
- `*retrospective` - Conduct retrospective
- `*daily-standup` - Run daily standup
- `*help` - Show all SM commands
- `*exit` - Exit SM mode

## When to Use SM Agent

- Creating stories
- Facilitating ceremonies (standup, retrospective, planning)
- Team process improvement
- Sprint management
- Agile coaching

## Related Agents

- **@pm** - Project planning
- **@po** - Backlog management
