This command activates the Full Stack Developer agent with complete persona and development guidelines.

## Quick Start

```
/dev
```

The dev agent will:
1. Load complete persona definition from agent file
2. Display adaptive greeting based on session context
3. Show quick commands for story development
4. Await your development requests

## Available Commands (In Dev Mode)

- `*develop` - Implement story tasks
- `*create-service` - Scaffold new service
- `*run-tests` - Execute linting and tests
- `*apply-qa-fixes` - Apply QA feedback
- `*waves` - Analyze parallel execution
- `*help` - Show all dev commands
- `*exit` - Exit dev mode

## When to Use Dev Agent

- Implementing user stories
- Debugging code issues
- Refactoring and optimization
- Running tests and validations
- Registering technical debt
