## Quick Start

```

```

## Available Commands (In Analyst Mode)

- `*analyze-requirements` - Analyze business requirements
- `*market-research` - Conduct market research
- `*competitor-analysis` - Analyze competitors
- `*create-prd` - Create product requirements document
- `*help` - Show all analyst commands
- `*exit` - Exit analyst mode

## When to Use Analyst Agent

- Requirements elicitation and analysis
- Market research
- Competitive analysis
- Business case analysis
- Feasibility studies
- Requirements documentation
