# /aios-master — Activate AIOS Master Orchestrator

**Agent:** @aios-master (Orion)

**File:** `.claude/commands/AIOS/agents/aios-master.md`

This command activates the AIOS Master agent for framework orchestration and comprehensive multi-domain expertise.

## Quick Start

```
/aios-master
```

or use the shorthand:

```
/AIOS
```

## Available Commands (In Master Mode)

- `*help` - Show all available agents and commands
- `*create-story` - Create new development story
- `*task {name}` - Execute specific framework task
- `*workflow {name}` - Run framework workflow
- `*kb` - Access AIOS Knowledge Base
- `*create-squad` - Assemble multi-agent team
- `*exit` - Exit master mode

## When to Use AIOS Master

- Orchestrating complex multi-agent workflows
- Creating framework components (agents, tasks, workflows)
- Accessing complete AIOS knowledge base
- Managing projects that span multiple domains
- Assembling and coordinating agent teams
- When no specialized agent fits your need

## Agent Quick Access

From master mode, activate any agent directly:

```
/dev /qa /architect /analyst /pm /po /sm /data-engineer /devops /squad-creator /ux-design-expert
```

## Framework Resources

Master has access to all AIOS resources:

- **115+ Tasks** in `.aios-core/development/tasks/`
- **7 Workflows** in `.aios-core/development/workflows/`
- **20+ Checklists** in `.aios-core/product/checklists/`
- **50+ Templates** in `.aios-core/product/templates/`
- **Knowledge Base** in `.aios-core/data/aios-kb.md`

## Example Master Commands

```
*create-story           → Create new development story
*task db-apply-migration → Execute database migration task
*workflow brownfield-fullstack → Run full-stack enhancement workflow
*create-squad           → Assemble specialized agent team
*kb                     → Search AIOS knowledge base
```

## Core Principles

- **Universal Executor** - Direct access to all framework resources
- **Orchestrator** - Coordinates multi-agent workflows
- **Fallback** - Use when specialized agent doesn't fit need
- **Framework Developer** - Create/modify AIOS components
