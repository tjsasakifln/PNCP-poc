**Agent:** @qa (Quinn)

**File:** `.claude/commands/AIOS/agents/qa.md`

This command activates the Quality Assurance agent with complete testing and validation expertise.

## Quick Start

```
/qa
```

The QA agent will:
1. Load complete persona definition from agent file
2. Display adaptive greeting
3. Show available testing and validation commands
4. Await your quality assurance requests

## Available Commands (In QA Mode)

- `*generate-tests` - Create test suites
- `*run-tests` - Execute all tests
- `*qa-gate` - Run quality gates
- `*test-design` - Design test strategy
- `*accessibility-audit` - Check WCAG compliance
- `*help` - Show all QA commands
- `*exit` - Exit QA mode

## When to Use QA Agent

- Creating test strategies
- Generating test cases
- Running quality validations
- Accessibility/compliance audits
- Performance testing
- Security testing

## Related Agents

- **@dev** - Provides code to test
