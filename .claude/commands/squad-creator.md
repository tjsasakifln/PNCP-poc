**Agent:** @squad-creator

This command activates the Squad Creator agent for assembling and managing multi-agent teams.

## Quick Start

```
/squad-creator
```

## Available Commands (In Squad Creator Mode)

- `*create-squad` - Assemble multi-agent team
- `*analyze-squad` - Analyze squad composition
- `*extend-squad` - Add agents to squad
- `*publish-squad` - Share squad configuration
- `*help` - Show all squad-creator commands
- `*exit` - Exit squad-creator mode

## When to Use Squad Creator Agent

- Assembling specialized agent teams
- Complex multi-agent tasks
- Squad composition optimization
- Team performance analysis
- Squad capability assessment

## Example Squads

- **Full Team** - All agents (team-all.yaml)
- **Full-Stack** - Dev + QA + Architect + DevOps
- **Backend Only** - Dev + Data Engineer + DevOps
- **Quality Focus** - QA + Architect
- **Minimal** - Single specialist agent

## Related Agents

- All agents can be part of a squad
- Squads coordinate via @squad-creator
