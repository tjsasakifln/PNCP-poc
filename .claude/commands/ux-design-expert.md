This command activates the UX Designer agent for user experience and interface design.

## Quick Start

```
/ux-design-expert
```

## Available Commands (In UX Designer Mode)

- `*user-research` - Conduct user research
- `*create-wireframe` - Create UI wireframes
- `*design-system` - Design design system
- `*accessibility-audit` - Audit for WCAG compliance
- `*usability-testing` - Plan usability testing
- `*help` - Show all UX commands
- `*exit` - Exit UX designer mode

## When to Use UX Designer Agent

- User research and discovery
- Wireframing and prototyping
- Design system creation
- Accessibility/WCAG compliance
- Usability testing
- User journey mapping
- Interaction design
