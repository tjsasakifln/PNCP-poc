## Formato de Numero CNJ

```
NNNNNNN-DD.AAAA.J.TR.OOOO

Onde:
N = Numero sequencial (7 digitos)
D = Digito verificador (2 digitos)
A = Ano de ajuizamento (4 digitos)
J = Justica (1 digito): 1=STF, 2=CNJ, 3=STJ, 4=Federal, 5=Trabalho, 6=Eleitoral, 7=Militar, 8=Estadual, 9=Militar Estadual
T = Tribunal (2 digitos)
O = Origem (4 digitos)
```

Exemplo: `0012345-67.2024.8.26.0100` = Processo 12345 de 2024, Justica Estadual (8), TJ-SP (26), Foro Central (0100)
