# SEO Score Checklist (0-100)

Complete checklist used by the SEO squad to score a website. Each check maps to a specific point value.

---

## On-Page SEO (25 points)

### Meta Title (4 pts)
- [ ] Title tag present on every page (1 pt)
- [ ] Title length 50-60 chars / under 580px (1 pt)
- [ ] Focus keyphrase in title (1 pt)
- [ ] Unique title per page (no duplicates) (1 pt)

### Meta Description (3 pts)
- [ ] Meta description present on every page (1 pt)
- [ ] Length 120-160 characters (1 pt)
- [ ] Includes focus keyphrase + CTA (1 pt)

### Heading Structure (3 pts)
- [ ] Single H1 per page (1 pt)
- [ ] H1 contains focus keyphrase (1 pt)
- [ ] Logical hierarchy H1 > H2 > H3 (1 pt)

### Keyword Placement (4 pts)
- [ ] Keyphrase in first 100 words (1 pt)
- [ ] Keyphrase in URL slug (1 pt)
- [ ] Keyphrase in at least one H2 (1 pt)
- [ ] Keyphrase in image alt text (1 pt)

### Keyword Density (2 pts)
- [ ] Density between 1-2.5% (1 pt)
- [ ] Even distribution throughout content (1 pt)

### Readability (3 pts)
- [ ] Flesch Reading Ease > 60 (1 pt)
- [ ] < 25% sentences over 20 words (1 pt)
- [ ] < 10% passive voice (1 pt)

### Content Length (2 pts)
- [ ] Minimum 300 words per page (1 pt)
- [ ] Competitive with top 10 for target keyphrase (1 pt)

### Links (2 pts)
- [ ] At least 1 internal link per page (1 pt)
- [ ] At least 1 external link to authority source (1 pt)

### Social Meta (2 pts)
- [ ] Open Graph tags complete (og:title, og:description, og:image) (1 pt)
- [ ] Twitter Card tags present (1 pt)

---

## Technical SEO (20 points)

### Status Codes (4 pts)
- [ ] All pages return 200 OK (2 pts)
- [ ] No 4xx errors on internal links (1 pt)
- [ ] No redirect chains (max 1 hop) (1 pt)

### Broken Links (3 pts)
- [ ] No broken internal links (2 pts)
- [ ] No broken external links (1 pt)

### Canonicals (3 pts)
- [ ] Canonical tag on every page (1 pt)
- [ ] Canonical points to correct URL (1 pt)
- [ ] No conflicting canonicals (1 pt)

### Indexability (3 pts)
- [ ] Important pages are indexable (1 pt)
- [ ] robots.txt doesn't block important pages (1 pt)
- [ ] No accidental noindex on key pages (1 pt)

### Sitemap & Robots (2 pts)
- [ ] XML sitemap at /sitemap.xml (1 pt)
- [ ] robots.txt present and valid (1 pt)

### HTTPS & Security (2 pts)
- [ ] HTTPS enforced on all pages (1 pt)
- [ ] No mixed content (1 pt)

### Internal Linking (2 pts)
- [ ] No orphan pages (1 pt)
- [ ] Descriptive anchor text used (1 pt)

### Mobile (1 pt)
- [ ] Viewport meta tag present (1 pt)

---

## Schema / Structured Data (15 points)

### JSON-LD Presence (3 pts)
- [ ] Structured data present on pages (2 pts)
- [ ] JSON-LD format used (Google preferred) (1 pt)

### Correct Page Type (3 pts)
- [ ] Schema type matches page content (2 pts)
- [ ] Required properties present per type (1 pt)

### Required Properties (3 pts)
- [ ] No missing required fields (2 pts)
- [ ] No JSON syntax errors (1 pt)

### Organization Schema (2 pts)
- [ ] Organization schema on homepage (1 pt)
- [ ] Includes name, url, logo, sameAs (1 pt)

### Breadcrumb Schema (2 pts)
- [ ] BreadcrumbList on interior pages (1 pt)
- [ ] Correct hierarchy reflecting navigation (1 pt)

### Rich Result Eligibility (2 pts)
- [ ] At least one page eligible for rich results (1 pt)
- [ ] FAQ/HowTo/Product schema where applicable (1 pt)

---

## Content Quality / E-E-A-T (15 points)

### Trust Pages (4 pts)
- [ ] About page exists (1 pt)
- [ ] Contact page/info accessible (1 pt)
- [ ] Privacy Policy present (1 pt)
- [ ] Terms of Service present (1 pt)

### Author Attribution (3 pts)
- [ ] Author byline on content pages (1 pt)
- [ ] Author bio/page exists (1 pt)
- [ ] Author credentials visible (1 pt)

### Content Depth (3 pts)
- [ ] Content covers topic thoroughly (1 pt)
- [ ] Related subtopics addressed (1 pt)
- [ ] Competitive with top-ranking pages (1 pt)

### E-E-A-T Signals (3 pts)
- [ ] Evidence of experience (case studies, testimonials) (1 pt)
- [ ] Expertise demonstrated (credentials, depth) (1 pt)
- [ ] Trust signals present (HTTPS, citations, accuracy) (1 pt)

### YMYL Compliance (2 pts)
- [ ] YMYL content identified and flagged (1 pt)
- [ ] Higher E-E-A-T standards applied to YMYL (1 pt)

---

## Performance / Core Web Vitals (10 points)

### LCP (3 pts)
- [ ] LCP < 2.5 seconds (2 pts)
- [ ] LCP element optimized (preloaded, right format) (1 pt)

### INP (2 pts)
- [ ] INP < 200ms (2 pts)

### CLS (2 pts)
- [ ] CLS < 0.1 (1 pt)
- [ ] All images have width/height attributes (1 pt)

### Image Optimization (2 pts)
- [ ] Images use modern format (WebP/AVIF) or compressed (1 pt)
- [ ] Below-fold images use lazy loading (1 pt)

### Font Loading (1 pt)
- [ ] font-display: swap or font preloading (1 pt)

---

## AI Visibility / GEO (10 points)

### llms.txt (2 pts)
- [ ] llms.txt present at site root (1 pt)
- [ ] All key pages listed with descriptions (1 pt)

### Statistics & Citations (2 pts)
- [ ] Content includes quantitative data (1 pt)
- [ ] Citations from credible sources present (1 pt)

### Self-Contained Content (2 pts)
- [ ] Content blocks make sense independently (1 pt)
- [ ] No "as mentioned above" dependencies (1 pt)

### Semantic HTML (2 pts)
- [ ] Proper heading hierarchy (1 pt)
- [ ] Semantic elements used (article, section, nav) (1 pt)

### AI Crawler Policy (1 pt)
- [ ] robots.txt includes AI crawler directives (1 pt)

### Direct Answers (1 pt)
- [ ] FAQ or Q&A formatted content present (1 pt)

---

## Site Architecture (5 points)

### URL Structure (2 pts)
- [ ] Clean, descriptive URLs with hyphens (1 pt)
- [ ] Logical hierarchy reflecting content (1 pt)

### Internal Linking (2 pts)
- [ ] Related content cross-linked (1 pt)
- [ ] No pages deeper than 3 clicks (1 pt)

### Navigation (1 pt)
- [ ] Navigation reflects content structure (1 pt)
