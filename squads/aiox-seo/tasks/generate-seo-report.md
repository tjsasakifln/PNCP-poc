# generate-seo-report

## Task: Generate Before/After SEO Improvement Report

### Metadata
- **executor:** seo-chief
- **depends_on:** evaluate-seo (before), optimize-seo, evaluate-seo (after)
- **elicit:** false
- **output:** seo-report.md

### Execution Steps

#### Step 1: Collect Data
- Load before-audit scores (from first evaluate-seo run)
- Load after-audit scores (from re-evaluation)
- Load changes manifest (from optimize-seo)

#### Step 2: Calculate Deltas
For each category and total:
- delta = after_score - before_score
- percentage_improvement = (delta / max_possible_improvement) * 100

#### Step 3: Generate Report

### Veto Conditions
- Cannot generate report without both before AND after scores
- Cannot claim improvements that didn't happen
- After score cannot be lower than before without explanation

### Completion Criteria
- Before/after comparison complete for all 7 categories
- All changes listed with specifics
- Manual recommendations clearly separated
- Report is self-contained and readable
